###########
Apuntes PIP
###########

pygraphviz
**********

Útil para generar gráficos de los modelos con `django-extensions <https://github.com/django-extensions/django-extensions>`_

.. code-block:: bash

    dnf install -y graphviz graphviz-devel

    (virtualenv) pip install pygraphviz
